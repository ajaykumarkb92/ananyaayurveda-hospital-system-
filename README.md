# Ayurvedic Hospital Management System

A complete patient management system for Ayurvedic hospitals with case study tracking.

## Features

- 🔍 Search patients by ID or name
- 🔎 Search across all symptoms and prescriptions
- 📝 Register new patients with auto-generated IDs
- 📚 Multiple case studies per patient
- 🏥 Department categories (Ortho, Nephro, Cardio, etc.)
- 💾 Browser-based storage (no server needed)
- 📱 Responsive design for mobile and desktop

## How to Use

1. Open index.html in any modern web browser
2. Search for existing patients or register new ones
3. Add case studies with symptoms and prescriptions
4. Search across all patient records using keywords

## Technology Stack

- Frontend: AngularJS 1.8.2
- Storage: IndexedDB (browser-based)
- Styling: Pure CSS with responsive design

## Future Cloud Migration

The system is designed to easily migrate to cloud databases (PostgreSQL, MySQL) when needed.
